# Dogs_demo

Le projet est un demo en flutter. Il est réalisé par deux différents architectures: **Provider** et **BLoC**

 
Avant de lancer le projet, exécutez cette commande pour installer les librairie:
```
flutter pub get
```

Lancer le projet:
```
flutter run
```

Lancer le test:
```
flutter test
```
